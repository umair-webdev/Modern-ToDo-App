## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete individual tasks
- 🔍 Filter tasks by **All**, **Active** and **Completed**
- 🧹 Clear all completed tasks
- 💾 Automatically saves tasks using **LocalStorage**
- 🌙 Dark mode and ☀️ Light mode
- 📱 Fully responsive design for mobile and desktop
- ⌨️ Add tasks quickly using the **Enter key**
- 🔢 Active task counter

## 💡 How It Works

Users can create tasks, mark them as complete, delete them and filter the task list. The application stores task data in the browser using LocalStorage, so tasks remain available even after refreshing or reopening the page.

The selected light or dark theme is also saved in LocalStorage.
